# MAD-Crew Project

MAD-Crew est un système automatisé de génération de projets utilisant des agents LLM orchestrés en équipes (crews).

## Installation

### Prérequis

- Python 3.9+
- Conda (recommandé)

### Environnement

```bash
conda create -n mad_crew python=3.10
conda activate mad_crew
pip install -e .
